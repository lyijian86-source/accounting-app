# 统计曲线重做设计

日期：2026-04-28

## 背景

当前统计页已经具备分类趋势和明细能力，但图形仍然是柱状条形，视觉气质偏工具感，不能支撑“高端金融看板感”的目标。用户明确希望所有数据呈现更极简、更高级，尤其是曲线表达。

## 目标

本轮只解决一个问题：把统计页里的趋势图从条形视觉重做为克制、金融感、更适合手机的极简曲线系统。

## 视觉方向

采用“曲线 + 轻面积填充”的金融看板方向：

1. 总览卡片里的 mini trend 改为超细曲线。
2. 明细区按天和按周趋势都改为主曲线视图。
3. 曲线下加极轻透明面积，增强质感和层次。
4. 默认不展示所有节点，只突出峰值点和最后一个点。
5. 保留极轻水平参考线，不做厚重坐标轴。

## 信息规则

1. 总览卡片仍以分类、变化率、变化额、占比为主，mini chart 只负责传达趋势轮廓。
2. 明细图成为主视觉，指标区退到辅助层。
3. 时间刻度只保留必要信息，避免图下方拥挤。

## 范围

本轮只做：

1. 重写总览卡片 mini trend。
2. 重写按天和按周趋势图。
3. 调整统计页图表配色、卡片层级和图形留白。

本轮不做：

1. 多曲线叠加。
2. hover 提示。
3. 缩放和拖拽。
4. 引入重型图表库。
5. 复杂动画。
